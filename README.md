# Simple Tools

A collection of simple, practical tools for everyday tasks.

## Overview

This repository contains a growing collection of lightweight, easy-to-use tools designed to solve common problems. Each tool is self-contained and focused on doing one thing well.

## Live Demo

Visit our [GitHub Pages site](https://liuchaoqun.github.io/simple_tools/) to access all available tools.

## Available Tools

Currently, this repository is being set up. Tools will be added soon!

## Planned Tools

Here are some tools we're planning to add:

- **Text Utilities**: String manipulation, encoding/decoding, formatting
- **Data Converters**: JSON/CSV/XML converters, unit converters
- **Generators**: UUID generator, password generator, QR code generator
- **Calculators**: Time zone converter, date calculator, math utilities
- **Developers Tools**: Base64 encoder/decoder, hash generator, regex tester

## Contributing

Contributions are welcome! If you have ideas for simple tools that would be useful:

1. Fork the repository
2. Create a new branch for your tool
3. Implement your tool following the existing structure
4. Submit a pull request
